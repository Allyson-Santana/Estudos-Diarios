    padronizada pela Ecma International na especificação ECMA-262 e ISO/IEC 
}

Sobre o javascript: {
    É uma linguagem interpretada 
    Linguagem de tipagem fraca e dinâmica
    Função de primeira classe: {
        A função pode se atribuida a uma: 
        variavel, 
        estrutura de dados: dentro de arrays e objects,
        Pode se passadas por argumentos
        Retorna a outras função
    }
    Closure: {
        Capacidade de uma variavel lembrar do ambiente que ela foi criada.
        Espoco de GLOBAL, Espoco de FUNÇÂO, Espoco de BLOCO
    },
    Currying: {
        Possibilita a função passar parametros que retorna outra função pelos proprios parametros
    }
    Hoisting: {
        Possibilita você chamar uma função declarada depois do Espoco
    }
    Imutabilidade: {
        Imutabilidade significa que a mudança não é feita no mesmo objeto, estrutura, 
        mas a mudança é representada em um novo. Cada mudança cria um novo valor e não toca no antigo.
    }
}
